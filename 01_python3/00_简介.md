python的运行
- 解释执行器类型：
```
CPython（C语言开发）
PyPy（python语言开发）
Jython（Java语言开发）
IronPython（.net开发）
```

- 常用程序文件的后缀：
```
python 文件后缀为: .py

其他：
.c   C语言源代码
.cpp/.cxx/.cc   C++语言源代码
.java   Java语言源代码
```

- 第一个Python程序
```
编写hello.py，在文件内写入一行：
print（“hello world！”）
执行python程序：
$ python3 hello.py
在hello.py，最开始写入 #!/usr/bin/env python3 后可以用 ./hello.py 的形式执行
```

- 在交互提示模式下编写并执行代码
```
进入交互模式：
$ python3 <回车>
退出交互模式：
>>> quit() <回车>
>>> exit() <回车>
输入：ctrl + d (输入文件结束符)
```
---

python 的注释：
```
python注释是以          # 开头，直至行尾
作用：让解释执行器忽略掉  # 号之后的内容，让其不参加执行
```
---

python 程序的组成：
```
1、程序由模块组成
2、模块由语句，函数，类组成
3、语句包含表达式
4、表达式是用来建立并处理数据对象
```
